Expense Tracker 
Overview
The Expense Tracker is a simple command-line Python application that helps users record, view, and calculate their daily expenses. It allows users to add expenditures with categories, list all recorded expenses, and calculate the total spending—all directly in the terminal.
Features
•	Add expenses with amount and category
•	Display a list of all expenses in a clean tabular format
•	Compute total amount spent
•	Simple menu-driven command-line interface
•	Lightweight and easy to run
Technologies / Tools Used
•	Python 3.x
•	Basic Python modules (no external libraries required)
•	Command-line interface (CLI)
Steps to Install & Run the Project
1.	Install Python 3 on your system.
2.	Save the project code into a file named `expense_tracker.py`.
3.	Open a terminal in the project folder.
4.	Run the project using `python expense_tracker.py` or `python3 expense_tracker.py`.
Instructions for Testing
5.	Add an expense by entering amount and category.
6.	Show expenses and ensure all entries appear correctly.
7.	Check the total and verify it matches the sum of added expenses.
8.	Exit the application using option 4.
